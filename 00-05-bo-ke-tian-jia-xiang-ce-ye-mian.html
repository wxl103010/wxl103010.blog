博客添加相册页面

hexo matery 博客搭建
博客搭建
发布日期:   2019-12-21
更新日期:   2020-01-01

说明

这次是添加单独的界面了，与之前的不一样，之前的界面是已经做好的，而现在所需要的页面就得根据自己的情况来创作了。根据网上同仁的资料，根据博客友情链接方式，以json管理做出界面，现整理以下做法。

原文章：https://liyangzone.com/2019/07/22/hexo%E5%8D%9A%E5%AE%A2%E6%B7%BB%E5%8A%A0%E4%B8%80%E7%BA%A7%E5%88%86%E7%B1%BB%E7%9B%B8%E5%86%8C/#toc-heading-1

添加相册界面

1、添加菜单

在博客配置文件_config.yml中添加以下信息。

  音乐:
    url: /musics
    icon: fas fa-music

注意：如果写好后报站，将每一行前面空的部分删除，用空格填充。

在博客主题中的 languages 文件夹下，将 default.yml 和 zh-CN.yml 文件，分别添加如下内容：

galleries: galleries
galleries: 相册

在主题目录的 layout/_partial/navigation.ejs 和 layout/_partial/mobile-nav.ejs 文件中均添加：

menuMap.set("Galleries", "相册");

2、生成文件

使用命令 hexo new page galleries生成相册页，打开刚生成的文件 /source/galleries/ 中的 index.md ，补全如下面的信息：

---
title: categories
date: 2019-12-19 01:53:52
type: "categories"
layout: "categories"
---

在主题目录中的 layout 目录下创建 galleries.ejs 文件，添加以下代码。

<link rel="stylesheet" href="/css/gallery.css">

<%- partial('_partial/bg-cover') %>

<main class="content">
    <div class="container">
        <% if (site.data && site.data.galleries) { %>
        <% var galleries = site.data.galleries; %>
        <div class="gallery-wrapper row">
            <% for (var i = 0, len = galleries.length; i < len; i++) { %>
            <% var gallery = galleries[i]; %>
            <div class="col s6 m4 l4 xl3 gallery-box">
                <a href="./<%- gallery.name %>" class="gallery-item" data-aos="zoom-in-up">
                    <div class="gallery-cover-box" style="background-image: url(http://localhost/<%- gallery.cover%>);">
                    </div>
                    <p class="gallery-name">
                        <%- gallery.name %>
                    </p>
                </a>
            </div>
            <% } %>
        </div>
        <% } %>
    </div>
</main>

还是该目录下，继续新建文件 galler.ejs ，写如下代码。

<link rel="stylesheet" href="/css/gallery.css">
<link type="text/css" href="/libs/fancybox/jquery.fancybox.css" rel="stylesheet">
<link type="text/css" href="/libs/justifiedGallery/justifiedGallery.min.css" rel="stylesheet">

<%- partial('_partial/post-cover') %>
<%
let galleries = [];
if (site.data && site.data.galleries) {
    galleries = site.data.galleries;
}
var pageTitle = page.title;
function getCurrentGallery(galleries, pageTitle) {
    for (let i = 0; i < galleries.length; i++) {
        if (galleries[i]['name'] == pageTitle) {
            return galleries[i];
        }
    }
}
var currentGallery = getCurrentGallery(galleries, pageTitle)

var photos = currentGallery.photos;

let imageStr = ''

for (var i = 0, len = photos.length; i < len; i++) {
    var photo = photos[i];

    imageStr += "<a href=\"http://localhost" + photo + "\"" +
            "     class=\"photo-item\" rel=\"example_group\"" +
            "     data-fancybox=\"images\">" +
            "      <img src=\"http://localhost" + photo + "\"" +
            "       alt=" + photo + ">\n" +
            "    </a>"

}
%>

<div class="container">
    <div class="photo-wrapper">
        <div class="waterfall" id="mygallery">
            <%- imageStr %>

        </div>
    </div>
</div>


<script src="/libs/fancybox/fancybox.js"></script>
<script src="/libs/justifiedGallery/justifiedGallery.min.js"></script>
<script>

  $("a[rel=example_group]").fancybox();
  $("#mygallery").justifiedGallery({margins: 5, rowHeight: 150});

</script>

以上代码说明：

上述代码的功能是读取相册配置文件，并把相册目录和相册列表都渲染成HTML，用<% %>包起来的代码是ejs语法。

上面代码里引用了两个jquery插件，分别是fancybox和justifiedGallery, 一个是点击弹出的轮播插件，一个是自动调整图片布局的插件，需要自行下载并放到相应目录，当然，你也可以用浏览器调试工具直接在我的博客上下载，在sources里找到对应的文件，另存为就行。

单独说明：

按上述方式，确实能添加相册页面，只是图片一直出不来，所以我个人将上面的图片链接代码稍作调整，路径改为localhost，也就是网址本身的地址，如果图片在博客目录下而非第三方，这样是可以的，如果是在第三方，那就需要根据第三方的链接而进行调整。

别急！下面还有些操作，操作完毕后，便可看到全部效果。

5、创建相片页面

在博客目录 sources/galleries 中新建与相册名称对应的文件夹，然后在文件夹中创建 index.md 文件，内容的话适当改变就可以，如下示例。

---
title: 2017
date: 2019-12-28 23:19:24
layout: "gallery"
---
